### Sub-Apps

- So this is a folder for sub app. Sub apps like : login page, checkout page, homepage etc all can be made as sub apps.
- how to create subapps : python manage.py startapp subapp #subapp is a name u can put any good name here.
- subapps may not have all same file as our main outdoor app but we can create them manually when needed.
- the main project never had any idea like we have sub-app we need to connect them.

### Step 1

- connect main project with the sub app.
- go to main settings.py change there : see for installed_app section and add sub app name.
